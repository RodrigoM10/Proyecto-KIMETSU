# ⚔️ Kimetsu no Yaiba - Personajes

¡Bienvenido/a al proyecto **Kimetsu no Yaiba - Personajes**! Esta es una aplicación web estática, sencilla y altamente visual desarrollada exclusivamente con **HTML5** y **CSS3**. 

El objetivo principal de este proyecto es presentar de manera limpia, atractiva y organizada a los cazadores de demonios, pilares y personajes principales del popular anime *Kimetsu no Yaiba* (Demon Slayer).

---

## 🚀 Características del Proyecto

*   **Diseño Temático:** Interfaz moderna con una paleta de colores y tipografía inspiradas en la estética del anime.
*   **Tarjetas de Personajes (Cards):** Cada personaje cuenta con su propia tarjeta contenedora que incluye su imagen, nombre completo, afiliación (Cuerpo de Cazadores, Demonio, etc.) y su respectivo estilo de respiración o habilidad de sangre.
*   **Efectos Visuales:** Transiciones y efectos de *hover* suaves con CSS para mejorar la interactividad del usuario al pasar el cursor sobre las tarjetas.
*   **Diseño Adaptable (Responsive Design):** Estructura fluida mediante el uso de layouts modernos (como Flexbox o CSS Grid) para asegurar que la página se vea perfecta tanto en dispositivos móviles, tablets como en pantallas de escritorio.
*   **Carga Eficiente:** Código nativo y optimizado, sin frameworks ni librerías pesadas de terceros.

---

## 🛠️ Tecnologías Utilizadas

*   **HTML5:** Para la estructuración semántica del contenido (encabezados, secciones, main, tarjetas, etc.).
*   **CSS3:** Para la estilización, maquetación adaptativa, colores, fuentes personalizadas y animaciones sutiles.
